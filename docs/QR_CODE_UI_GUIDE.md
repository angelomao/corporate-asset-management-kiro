## Color Scheme

### Buttons
- **Primary (Scan QR, Generate)**: Green (#10B981)
- **Secondary (Download)**: Blue (#3B82F6)
- **Success (Print)**: Green (#10B981)
- **Danger (Cancel, Close)**: Gray (#6B7280)

### Status Indicators
- **Success**: Green background with checkmark
- **Error**: Red background with X
- **Warning**: Yellow background with !
- **Info**: Blue background with i

### QR Code Display
- **Background**: White (#FFFFFF)
- **Border**: Gray (#E5E7EB)
- **QR Code**: Black on white for maximum contrast

This UI guide provides a visual reference for all QR code-related interfaces in the system. Use it to understand the user experience and design consistency across the feature.
